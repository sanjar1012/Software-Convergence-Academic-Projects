# Campus Sustainability & Urban Analytics AI Suite

This repository contains two core undergraduate research projects exploring machine learning workflows and software convergence applications, developed for smart city and campus infrastructure.

## Project Structure
* `waste_classification.py`: A PyTorch-based Deep Learning pipeline utilizing transfer learning (ResNet-50) for automated multi-class recyclable material sorting.
* `traffic_pipeline.py`: A TensorFlow/Keras backend pipeline using an optimized MobileNetV2 architecture for real-time traffic density estimation and anomalous event detection.
